Nombre app: APP Pokémon

Descripcion:Este proyecto es una aplicación móvil desarrollada en React Native que permite a los usuarios buscar y explorar información sobre diferentes Pokémon. Utilizando la PokeAPI, la aplicación ofrece funcionalidades como la visualización de la lista de Pokémon, la consulta de sus detalles y la visualización de sus evoluciones.

Integrantes: Ramiréz Flores Kevin, Fuentes Osorio Cristian

Dependencias: npm install react-native react-navigation react-navigation-stack react-navigation-tabs @react-native-community/viewpager react-native-gesture-handler react-native-reanimated react-native-safe-area-context react-native-screens react-native-vector-icons react-navigation-drawer react-native-datepicker
